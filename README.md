# LINES

スマホブラウザとPWAを前提にした、テトリス系パズルのプロトタイプです。

## 起動

```powershell
node dev-server.mjs
```

ブラウザで `http://127.0.0.1:4173` を開きます。

## 構成

- `index.html`: DOM UI。タイトル、モード選択、HUD、オプション。
- `styles.css`: スマホ縦画面向けの見た目と安全領域対応。
- `app.js`: Canvas描画、テトリス風の盤面ロジック、入力処理。
- `manifest.webmanifest`: PWAのアプリ名、表示モード、アイコン。
- `sw.js`: オフライン起動用のService Worker。
- `dev-server.mjs`: 依存なしの開発用ローカルサーバー。

## 関連ドキュメント

- `GAME_DESIGN.md`: モード、操作、速度、Blast、Clean Upなどのゲーム仕様。
- `ACHIEVEMENTS.md`: 実績一覧、カウント条件、保存データ。
- `RELEASE_CHECKLIST.md`: 更新前後の確認手順、PWA/スマホ確認項目。

## 参考データを読ませるなら

まずは、参考にしたいゲームのスクリーンショット、動画、メモ、仕様らしきテキストを `references/` フォルダに入れるのが扱いやすいです。

どこが該当部分かわからない場合でも、以下のような粒度で十分です。

- 画面遷移がわかるスクリーンショット
- 操作感がわかる短い動画
- ステージ、スキル、アイテム、敵、ミッションなどのメモ
- 似せたいUIや避けたいUIの画像

読み取った内容は、主に `app.js` の `modes`、盤面ロジック、入力処理、`index.html` のメニュー構造へ反映していきます。
